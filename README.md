A colorful, interactive quiz application built with HTML, CSS, and JavaScript.

Quiz App is a highly interactive, colorful, and user-friendly web application built using HTML, CSS, and JavaScript. It is designed to provide an engaging learning experience where users can test and improve their knowledge across multiple domains such as General Knowledge, Programming, and Mathematics. The app dynamically loads multiple-choice questions and provides instant feedback for every answer, helping users learn from their mistakes in real time.

The application features a countdown timer for each question, creating a fun and challenging environment while encouraging quick thinking. If the user answers incorrectly or fails to respond within the allotted time, the app displays the correct answer, turning every quiz attempt into a learning opportunity. Users can track their current score, choose to show or hide it, and their best scores are saved locally using localStorage, providing a sense of achievement and motivation to improve.

In addition to its functional features, the app emphasizes modern UI design and usability. The quiz interface includes colorful gradient buttons, a semi-transparent quiz container to make content readable over the background, and a quiz-themed seamless background pattern that enhances the overall aesthetic. Users can also toggle between light and dark themes, making the app visually comfortable in different lighting conditions. Navigation is intuitive, with options to restart a quiz or go back to the category selection, ensuring a smooth user experience from start to finish.

Overall, Quiz App is a fun, educational, and visually appealing tool for learners of all ages to test their knowledge, improve their learning skills, and enjoy a modern quiz-taking experience directly in the web browser without any external dependencies or complex setups.

#Features
*Choose from three quiz categories: GK, Programming, Maths
*10 questions per quiz, loaded dynamically
*Timer countdown for each question
*Instant feedback: correct/incorrect and shows correct answer if wrong or time’s up
*Score display and toggle
*Best score saved using localStorage
*Restart and Back to Category buttons
*Dark/Light theme toggle
*Colorful and responsive UI with gradient buttons and background image

#Technologies Used
*HTML5
*CSS3
*JavaScript (vanilla)

#How to Use
*Open index.html in your browser.
*Select a quiz category.
*Answer the questions within the time limit.
*View instant feedback and track your score.
*Toggle dark/light theme or restart quiz anytime.
